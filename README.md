## Mô hình YOLOv10
Mô hình YOLOv10 được huấn luyện để nhận dạng 5 loại bệnh trên lá cây cà chua:
- **Early blight**
- **Late blight**
- **Leaf mold**
- **Yellow leaf curl virus**
- **Lá khỏe mạnh**
