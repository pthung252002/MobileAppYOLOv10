<h1 align="center">🍅 Tomato Leaf Disease Detection with YOLOv10 🍅</h1>

## YOLOv10 Model  
The YOLOv10 model has been trained to detect 5 types of tomato leaf diseases:  
- **Early blight**  
- **Late blight**  
- **Leaf mold**  
- **Yellow leaf curl virus**  
- **Healthy leaves**
